jsNIhonIME - Javascript Japanese Input

Type inside the text box and the roman characters will be replaced by their correspondent kana.
You may use lower case for hiragana and lower case for katakana.

Some characters are accepted by using more than one romanization:
  • し, shi, also accepted as [si]
  • じ, ji, also accepted as [zi]
  • ぢ, 'ji, also accepted as [di]
  • づ, 'zu, also accepted as [du]
  • ん, n, is detected as you type n followed by a consonant, or at the end of a word (space). Another option is to use'n, n', ñ or ç.
  • っ, little tsu(shows a double consonant), is detected as you type t followed by a consonant, or if you repeat a consonant.
The same goes for their katakana counterparts.

It is also possible to type some punctuation, that will be replaced by their full-width equivalents.

I also implemented buttons as a way to input hiragana, katakana and each level of the jouyou kanji.
If you keep your cursor over each button, its meaning and readings(japanese, kun-yomi, and chinese, on-yomi) will appear.



In order to save the text, you only have to copy it to a text editor with a font that supports Unicode(i.e Arial Unicode).
